# Simple Task Manager on Laravel

## General info
This is a primitive task manager for you to create tasks, task statuses, and task labels. You may create a task, set a person who is responsible for doing it, choose status, and add a label. You may see all the tasks created on the special page. You can edit the task and delete it (note, that some actions are restricted for unauthorized users and for a user who is not the taskâ€™s creator). App works on the standart php server. You haven't use additional software.

Public app link - https://web-production-0da7.up.railway.app/

## Technologies
Project is created with:
* PHP: 8.0.2
* Laravel: 9.39
* Bootstrap: 4.0

* laravelcollective/html
* rollbar/rollbar-laravel
* spatie/laravel-query-builder

## Setup

```sh
$ git clone https://github.com/IgBuS/php-project-lvl4.git

$ make setup
```

## Start

```sh
$ make start
```
